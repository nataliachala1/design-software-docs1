Este directorio contiene los artefactos de control del proyecto **Horarios SENA**. Aquí se documentan los pendientes técnicos, los riesgos identificados, las dependencias del proyecto y las preguntas abiertas que requieren decisión.

Estos documentos permiten al equipo tener una visión clara del estado del proyecto, los riesgos que pueden afectar el desarrollo y las decisiones que aún están pendientes. Se actualizan periódicamente durante el desarrollo del proyecto.
