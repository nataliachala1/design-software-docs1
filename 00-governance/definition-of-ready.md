# Definition of Ready

> Estado: 🟢 Estable | Última actualización: 2026-06-24
> Autor: ADSO 314555 | Equipo: Arquitectura

Un documento está **listo para revisión** cuando cumple todos los criterios. Si falla uno, el PR no se abre.

## Criterios

- [ ] Tiene título, estado, fecha y autor o equipo responsable.
- [ ] Las secciones `Contexto` y `Contenido` tienen texto real — no vacías ni con comentarios placeholder.
- [ ] Está enlazado desde el `README.md` de su sección.
- [ ] Los diagramas referenciados existen en `08-uml/` con fuente editable.
- [ ] No contiene credenciales, tokens, datos personales ni información sensible.
- [ ] El autor lo revisó antes de abrir el PR.

## Regla

> Un documento que no cumple el DoR se devuelve sin revisión de contenido. El revisor solo verifica el checklist.
