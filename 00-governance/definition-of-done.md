# Definition of Done

> Estado: 🟢 Estable | Última actualización: 2026-06-24
> Autor: ADSO 314555 | Equipo: Arquitectura

Un documento está **done** cuando cumple todos los criterios. El merge no se hace hasta que el checklist esté completo.

## Criterios

- [ ] Aprobado por al menos un revisor distinto al autor.
- [ ] Enlazado desde el `README.md` de su sección.
- [ ] Estado cambiado a 🟢.
- [ ] Todos los enlaces relativos funcionan.
- [ ] No contiene información sensible.
- [ ] `CHANGELOG.md` actualizado si el cambio afecta gobernanza, estructura de carpetas o un contrato compartido (API, modelo de datos, convenciones de Git).

## Regla

> Sin aprobación de un revisor externo al autor, el estado no puede ser 🟢 y el PR no se mergea.
