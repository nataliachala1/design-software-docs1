## Principios de capacitación

1. **Claridad:** Los manuales deben ser claros y fáciles de entender.
2. **Paso a paso:** Las instrucciones deben ser secuenciales y accionables.
3. **Ejemplos:** Incluir ejemplos prácticos de uso.
4. **Imágenes:** Usar capturas de pantalla cuando sea posible (en la versión final).
5. **Accesibilidad:** El lenguaje debe ser accesible para usuarios no técnicos.
