## Principios de diseño UX/UI

| Principio | Descripción |
|-----------|-------------|
| **Consistencia** | Los elementos visuales y de interacción son consistentes en toda la aplicación. |
| **Claridad** | La información y las acciones son claras y fáciles de entender. |
| **Eficiencia** | Los flujos de trabajo están optimizados para minimizar pasos innecesarios. |
| **Retroalimentación** | El usuario recibe feedback claro para cada acción (éxito, error, carga). |
| **Accesibilidad** | La aplicación cumple con WCAG 2.1 AA para usuarios con discapacidades. |
| **Responsive** | La interfaz se adapta a diferentes tamaños de pantalla (mobile, tablet, desktop). |
